# Compress Comic

`compress-comic` 是一个使用 Go 语言编写的自动化漫画压缩工具。它能够扫描指定目录下的章节文件夹，将其转换为压缩包（ZIP, 7z 或 RAR），并在验证压缩包完整性后自动删除原始文件夹。

## 功能特性

- **自动化处理**：自动遍历目录，识别仅包含图片的章节文件夹。
- **多格式支持**：支持输出为 `ZIP`、`7z` 和 `RAR` 格式。
- **目录平铺**：压缩包内直接存放图片，不包含额外的章节根目录。
- **并发处理**：支持多线程并行压缩，显著提高处理速度。
- **安全性保障**：内置压缩包完整性验证（使用 `7z t`），只有验证通过后才会删除原文件夹。
- **RAR 恢复记录**：支持为 RAR 格式添加自定义比例的恢复记录（防止文件损坏）。

## 前置要求

本工具依赖以下外部命令行工具：

1.  **7-Zip** (`7z`): 用于创建 ZIP/7z 压缩包以及进行所有格式的完整性验证。请确保 `7z` 已添加到系统环境变量 PATH 中。
2.  **WinRAR** (`rar`): 仅在需要输出 `RAR` 格式时使用。程序会自动尝试在默认安装路径查找，也可以通过 `-rar-path` 参数手动指定。

## 安装说明

如果你已经安装了 Go 环境，可以克隆本项目并编译：

```bash
go build -ldflags="-s -w" -trimpath -o compress-comic.exe main.go
```

或者直接运行项目中的 `build.bat`（仅限 Windows）。

## 使用方法

### 基本命令

在当前目录下创建一个名为 `comic` 的文件夹，放入你的漫画章节，然后运行：

```bash
./compress-comic.exe
```

### 命令行参数

| 参数 | 描述 | 默认值 |
| :--- | :--- | :--- |
| `-dir` | 漫画根目录路径（支持相对目录） | `pwd/comic` |
| `-workers` | 并发处理的工作线程数 | CPU 核心数 (最大 8) |
| `-format` | 输出格式 (`zip`, `7z`, `rar`) | `zip` |
| `-rr` | RAR 恢复记录百分比 (仅对 rar 有效) | `10` |
| `-rar-path` | 手动指定 `rar.exe` 的路径 | (自动查找) |
| `-v` | 显示详细的压缩和验证过程信息 | `false` |

### 示例

**1. 使用 4 个线程将目录下的章节压缩为 7z 格式：**

```bash
./compress-comic.exe -dir "D:\Comics\MyComic" -format 7z -workers 4
```

**2. 压缩为带有 5% 恢复记录的 RAR 格式，并显示详细日志：**

```bash
./compress-comic.exe -format rar -rr 5 -v
```

## 工作流程

1.  **扫描**：递归遍历根目录，寻找只包含图片（JPG, PNG, WEBP 等）且不含子目录的文件夹。
2.  **压缩**：调用 `7z` 或 `rar` 命令进行最高压缩率处理。
3.  **验证**：调用 `7z t` 测试压缩包是否损坏。
4.  **清理**：如果验证成功，删除源文件夹；如果失败，保留源文件并记录错误。
